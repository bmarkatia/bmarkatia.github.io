Project Description

Developed an inventory system using inheritance and polymorphism to efficiently track and manage product data. Applied encapsulation and modular design principles to enhance scalability and ease of maintenance. Built interactive classes and integrated file I/O to support adding, removing, and updating product records through a simple interface.

Skills Learned

  • Object-oriented programming with Java
  • Implementation of inheritance and polymorphism
  • Encapsulation and modular design principles
  • File I/O operations for data persistence
  • Building scalable and maintainable software systems
