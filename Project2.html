Project Overview

A MIPS processor simulation programmed in C, implementing core processor functions and a single-cycle datapath.

Project Description

Programmed core MIPS processor functions in C, including stack and queue operations for data manipulation. Debugged and optimized multi-threaded code to improve reliability and runtime performance. Implemented instruction decoding, register updates, and arithmetic operations to simulate a single-cycle datapath.

Skills Learned

  • C programming and systems software development
  • Understanding of processor architecture and MIPS instruction set
  • Implementation of stack and queue data structures
  • Multi-threaded programming and debugging
  • Code optimization for performance and reliability
  • Instruction decoding and register management
